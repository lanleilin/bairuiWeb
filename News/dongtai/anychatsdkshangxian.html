AnyChat SDK V5.3发布上线

发表日期：2015-07-28 14:20:02

AnyChat Platform Core SDK V5.3于7月15日正式发布。


新版本新增 “智能排队”功能和中心服务器拍照功能，优化完善Web平台功能、移动设备（android、ios）支持合成流录制等。


本次升级对之前的旧版本（包含V5.0及以上的版本）向下兼容，建议客户端、服务器均升级到最新版本以达到最佳效果。不过您可以根据实际情况，分阶段进行升级，比如先升级服务器到最新版本，然后逐渐升级Windows客户端、手机客户端等平台，新旧版本客户端以及新旧版本的服务器之间可以正常访问。


本次升级更新详细内容包括：

开放智能排队API接口，提供完整的业务排队解决方案

增加中心服务器拍照功能

支持标准SIP协议，可实现与硬件终端、第三方平台的互联互通

Web平台支持在视频上迭加文字、图片等内容，透明度可设置

Web平台提供将本地文件内容转换为BASE64编码字符串的能力

Web SDK支持禁止双击视频画面全屏显示

客户端合成录制以及合成流录制支持多路视频混合

支持动态调节视频参数，包括码率、视频质量、预设参数等

开放API接口可禁止录像文件按日期分目录保存

客户端SDK开放网络数据发送回调接口

支持Mic热拔插，修正热拔插Mic后音频偶尔失效的问题

修正使用域名连接服务器在某些情形下不能产生Connect异步事件的问题

针对移动终端网络信号不稳定导致视频长时间卡顿的问题进行优化

修正在android 5.0系统上应用闪退的问题

移动设备（Android、iOS）支持合成流录制

服务器开放新的API接口，支持获取房间列表及用户列表

修正Web平台重复调用设置背景图片API会导致内存泄露的问题

优化中心录像服务器，增强系统的稳定性，更详细的日志记录

优化核心服务器，降低高负载时对系统资源的需求

优化客户端本地录制，修正多次重复录制可能出现异常的问题

 

详细升级信息请点击：AnyChat Platform Core SDK V5.3
